# Component Playground

This is a development playground to test and preview all BSA Core UI components.

## Running the Playground

```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

## Available Examples

- **Layout** - Page containers, cards, and sections
- **Icons** - Icon components and usage
- **Buttons** - Button variants and styles
- **Inputs** - All input components (text, select, search, date)
- **Modal** - Modal dialogs and confirmations
- **Form** - Form components with validation
- **Table** - BsaDataTable with PrimeVue integration
- **Loading** - Loading indicators and spinners

## Adding New Examples

Create new example files in `playground/examples/` and import them in `playground/App.vue`.
